# Himnario ASD (v2.0.3)

## Descripción
- Aplicación para proyectar los 613 himnos en un formato de video de una manera fácil y rápida en un computadora o laptop.
- Puedes crear listas de reproducción para que la música siga sin interrupciones.
- Una aplicación ligera con un diseño minimalista, que le permite una acompañar sus servicios en la iglesia.
<br/>

## Características
Esta versión incluye los siguientes cambios:
- Contenido del nuevo himnario adventista en formato de video 
- Incluye los de los 613 himnos actualizados
- Funcionalidad para crear listas de reproducción de himnos
- Selección del tema de color (claro, oscuro)
- Funciones de búsqueda de himnos por palabras o número
- Controles de reproducción para cambio de pista (cantado o solo música), pausa, y mute
- Atajos del teclado para regresar la navegación y activar pantalla completa
- El título de la ventana cambia respecto al himno que se reproduce
<br/>

> [!IMPORTANT]
> **WINDOWS**

1. Para realizar la instalación en un equipo con Windows primero deberas descargar los tres (3) archivos comprimidos siguientes:
```sh
himnario_asd_2.0.3_compressed.7z.001
himnario_asd_2.0.3_compressed.7z.002
himnario_asd_2.0.3_compressed.7z.003
```
2. Lo siguiente es descargar el programa [7-zip](https://www.7-zip.org/) para descomprimir los archivos. Puedes descargarlo desde su sitio oficial [https://www.7-zip.org/](https://www.7-zip.org/) o dar click [aquí](https://www.7-zip.org/a/7z2501-x64.exe)
3. Una vez tengas descargado el instalador del programa [7-zip](https://www.7-zip.org/) deberas ejecutarlo
4. Ahora sobre el archivo que descargaste
```sh
himnario_asd_2.0.3_compressed.7z.001
```
da click derecho sobre el y selecciona en el menú desplegable
```sh
"7-Zip" > "Extraer aquí" or "Extraer a himnario_asd_2.0.3_compressed\\" 
```
para que el software [7-zip](https://www.7-zip.org/) pueda descomprimir correctamente el archivo
5. La carpeta resultante contiene los archivos necesarios para ejecutar el **Himnario ASD (v2.0.3)**
6. Dentro de la carpeta extraida puedes ejecutar el archivo
```sh
himnario_asd.exe
```
para inciar el programa
7. Puedes mover la carpeta a donde prefieras. La aplicación **Himnario ASD (v2.0.3)** es portable en su propia carpeta.
